# PRODUCT.md

## Product Name

codex-dev-harness

## Product Category

Agentic Development Repo Template / Governed Coding Workflow Template.

## Purpose

Provide a reusable documentation-first baseline for projects where AI/Codex helps inspect, modify, verify, and hand off software work.

## Target Users

- Solo developer using Codex for structured development work.
- Team using AI-assisted coding with explicit safety and verification rules.
- Project maintainer who wants repeatable handoff and closeout discipline.

## Core Value

This template makes AI-assisted development safer by combining task contracts, agent instructions, profile selection, side-effect boundaries, private-data protection, verification, and audit evidence.

## Non-Goals

- It is not an application framework.
- It is not a code generator in P0.
- It is not a replacement for project-specific architecture decisions.
- It does not include real business data, credentials, or device parameters.

## Product Principles

- Documentation before automation.
- One-agent-first before multi-agent orchestration.
- Read-only first before mutation.
- Dry-run before apply.
- Evidence before closeout.
