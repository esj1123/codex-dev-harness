# ROADMAP.md

## P0: Docs Only

Goal: establish the documentation baseline and template contract.

Done when:
- Core markdown files exist.
- docs/ policy files exist.
- templates/base placeholders exist.
- P0 scope and non-goals are explicit.

## P1: Base Templates

Goal: refine base templates into usable copy targets.

Done when:
- Base templates have stable placeholders.
- README and AGENTS templates are internally consistent.
- Acceptance trace template supports requirement-to-evidence mapping.

## P2: Profile Templates

Goal: add profile-specific template variants.

Candidate profiles:
- inspection.
- governed_coding.
- workflow_orchestration.
- multi_agent_lab.
- managed_enterprise.

## P3: Quality Gate Scripts

Goal: define and then implement minimal quality gate scripts.

P3 is not started in P0.

## P4: Render Script

Goal: render selected templates from a config file.

P4 is not started in P0.

## P5: Examples

Goal: validate the template against minimal example projects.

Candidate examples:
- python_cli_minimal.
- csharp_desktop_minimal.
- plc_tool_minimal.
- data_pipeline_minimal.

## P6: Release

Goal: tag a reusable baseline and document known limitations.
