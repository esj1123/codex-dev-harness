# AGENTS.override.md - csharp_desktop_minimal

## Purpose

Apply the csharp_desktop profile to a minimal example project.

## Expected Layout

- Solution/project files are intentionally absent in this P3 skeleton.
- build.ps1, test.ps1, and smoke.ps1 are intentionally absent.
- Desktop smoke behavior is intentionally undefined.

## Verification Defaults

- Build: NOT RUN until project files exist.
- Test: NOT RUN until test entrypoint exists.
- Smoke: NOT RUN until safe smoke steps exist.
