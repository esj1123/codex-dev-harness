# AGENTS.override.md - plc_tool_minimal

## Purpose

Apply the plc_or_device_tool profile to a minimal example project.

## Allowed Default Work

- Documentation.
- Simulator/mock assumptions.
- Read-only polling boundary notes.
- Safety and verification planning.

## Prohibited Default Work

- Live device write.
- Start, stop, reset, mode change, or control action.
- Real device connection code.
- Equipment IP, port, tag, address, or live parameter details.
- Secret or config generation.
