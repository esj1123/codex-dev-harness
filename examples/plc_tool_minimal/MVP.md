# MVP - plc_tool_minimal

## MVP Scope

- Generate control documents for a minimal PLC or device tool project.
- Make simulator/mock first explicit.
- Make live device write prohibited by default.
- Make equipment detail exclusion explicit.

## Out of Scope

- Device or PLC source code.
- Connection configuration.
- Live polling implementation.
- Start, stop, reset, mode change, or write action.
