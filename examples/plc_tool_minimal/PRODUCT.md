# PRODUCT - plc_tool_minimal

## Product Purpose

Show how an Agentic Development Repo Template can be applied to a minimal PLC or device tool shape with strong side-effect boundaries.

## Users

- Template maintainers validating device-tool safety defaults.
- AI coding agents checking equipment-adjacent guardrails before implementation.

## Non-Goals

- No live equipment control.
- No device connection.
- No equipment IP, port, tag, address, or live parameter details.
